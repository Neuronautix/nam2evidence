Readiness Tab
=============

The Readiness tab presents a multi-dimensional maturity assessment for the
standardized evidence package.

Purpose
-------

Readiness scoring helps prioritize cleanup. It is not a regulatory decision and
does not certify that evidence is acceptable.

Top score
---------

The top card shows:

* readiness label;
* total score;
* maximum score;
* percentage;
* disclaimer.

Dimensions
----------

Each dimension has:

* label;
* score and maximum score;
* visual progress bar;
* rationale;
* suggested improvement.

Validation summary
------------------

The page repeats validation error, warning, and blocking counts so readiness can
be interpreted alongside semantic validation.

Blocking gaps
-------------

Blocking gaps must be addressed before formal package use. Treat them as the
main action list.

Recommended improvements
------------------------

Improvements may not all be blocking, but they improve reproducibility,
interoperability, and review confidence.

How to use readiness
--------------------

Use readiness as a prioritization tool:

1. resolve blocking gaps first;
2. address low-scoring dimensions;
3. rerun validation and revisit readiness;
4. document residual limitations.
