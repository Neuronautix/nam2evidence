eCTD Mapping Tab
================

The eCTD Mapping tab proposes where evidence documents and claims belong in
eCTD Module 4 nonclinical sections.

Purpose
-------

This screen organizes evidence placement. It does not produce an official eCTD
submission and does not provide regulatory advice.

What you see
------------

The screen shows an eCTD-like Module 4 hierarchy. Sections with mapped evidence
are highlighted.

Add a mapping
-------------

1. Click **Add mapping**.
2. Select or enter the target module section.
3. Link relevant study IDs or claim IDs.
4. Add placement rationale.
5. Save.

Edit or delete a mapping
------------------------

Use the mapping controls in the tree. Deleting a mapping removes the proposed
placement only; it does not delete the study or claim.

Review checklist
----------------

* Does the document belong in the chosen Module 4 section?
* Is the rationale clear?
* Are linked claims already reviewed?
* Are caveats preserved?

Recommended use
---------------

Use this screen as a planning aid for regulatory operations staff. Final eCTD
placement should be reviewed by qualified regulatory professionals.
