Project Overview Tab
====================

The Overview tab is the project dashboard. It summarizes the project and gives
quick entry points to the main workspaces.

Top project summary
-------------------

The header shows:

* project name;
* project description;
* drug name;
* sponsor;
* review status.

Workspace cards
---------------

Cards link to:

* Context of Use;
* NAM Study;
* Import NAMO;
* Validation Matrix;
* Claim Graph;
* eCTD Mapping;
* Export Center.

How to use it
-------------

Use Overview as a checklist. If a card says "Not created" or shows incomplete
counts, open that workspace before attempting export.

What good looks like
--------------------

Before export, you should see:

* a Context of Use card;
* a NAM Study record;
* validation evidence reviewed;
* claims approved;
* eCTD mappings present when needed;
* no major blockers in NAM-CORE validation tabs.
