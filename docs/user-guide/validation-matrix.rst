Validation Matrix Tab
=====================

The Validation Matrix tab records evidence against an eight-domain
fitness-for-purpose framework.

Purpose
-------

This screen helps reviewers see whether the model and study are sufficiently
supported for the declared context of use.

Statuses
--------

Evidence items can be marked as:

* met;
* partial;
* not met;
* not applicable.

How to use
----------

1. Read each evidence item.
2. Check the evidence type and supporting data.
3. Set or update the status.
4. Use notes to understand unresolved gaps.

Review guidance
---------------

Do not mark an item as met because evidence exists somewhere. Mark it met only
if the evidence is relevant to the stated context of use and sufficient for the
criterion.

Export relationship
-------------------

Validation evidence appears in exported dossiers. Weak or incomplete validation
evidence should be made explicit rather than hidden.
