Glossary
========

.. glossary::

   AI-ready
      Structured enough to support downstream machine processing or analysis.
      This does not mean scientifically or regulatorily accepted.

   Context of Use
      The specific question, intended use, decision, biological domain, and
      limitations that define how evidence should be interpreted.

   CURIE
      Compact URI, such as ``CL:0002310``. Used to identify ontology terms.

   eCTD
      Electronic Common Technical Document. nam2evidence proposes evidence
      placement but does not create an official submission.

   FAIR
      Findable, Accessible, Interoperable, Reusable.

   NAM
      New Approach Methodology.

   NAM-CORE
      The proof-of-concept standardization layer used by nam2evidence for
      endpoint data, ontology mappings, semantic validation, provenance, and
      reusable exports.

   NAMO
      A NAM-oriented ontology/schema foundation used as an alignment target for
      model and study metadata.

   Ontology mapping
      Linking a source value to a controlled vocabulary term.

   Provenance
      Information about where data came from and how it was processed.

   Readiness score
      Internal maturity score that helps prioritize cleanup. It is not a
      regulatory determination.

   Review gate
      A condition that blocks formal export until a qualified person resolves or
      approves it.
