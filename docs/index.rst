nam2evidence User Documentation
================================

nam2evidence is a proof-of-concept application for organizing, standardizing,
reviewing, and exporting evidence from New Approach Methodologies (NAMs). It is
designed around a context-of-use question, a structured NAM study record, review
gates, NAM-CORE standardization workspaces, and reusable exports.

This documentation is written for two audiences:

* non-coders who need to start the app and use the screens safely;
* technical users who need operational commands, troubleshooting, and references.

Important boundary
------------------

nam2evidence standardizes and packages evidence. It does not validate a
scientific method, provide regulatory advice, create an official SEND/CDISC
submission, or guarantee acceptance by any authority. Qualified human review is
required before outputs are used in an IND, eCTD, or other filing.
